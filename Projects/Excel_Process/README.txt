Python version: 3.6.x

Library required: openpyxl

Library installation:
    pip install openpyxl|| pip3 install openpyxl

Check whether the library installed properly:
    pip list || pip3 list

What the scripts do:
    col_plus_one.py:
        Load the workbook
        Select the worksheet
	Iterate rows in column (Plus one)
    to_text.py:
        Load the workbook
        Select the worksheet
        Read data from Excel
    	Put it into text file
